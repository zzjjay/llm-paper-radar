针对结构化剪枝在 LLM/VLM 上精度损失大的问题，作者通过 virtual gate 机制分析发现深层 layer 对 decode（next-token prediction）至关重要，但对 prefill（context encoding）基本冗余。基于此提出 POP：prefill 阶段跳过深层 layer，decode 阶段保留完整模型。为保证 KV cache 一致性，引入独立的 KV projection，并设计边界处理策略以保证首个生成 token 的准确性。在 Llama-3.1、Qwen3-VL、Gemma-3 上实现最高 1.37× prefill 加速且性能损失极小。

- 💡 揭示深层 layer 对 prefill 冗余、对 decode 关键
- 🎯 prefill 跳层 + decode 全模型的 stage-aware 推理
- 🔧 独立 KV projection 保持 cache 一致性
- 📊 Llama-3.1/Qwen3-VL/Gemma-3 上 prefill 加速 1.37×
