#### 摘要
On-Policy Distillation (OPD) 训练不稳定的根源在于单样本 Monte Carlo 估计量的高梯度方差。本文将 OPD 重新建模为 policy-gradient RL，并引入 control variate baseline（即 value function）来降低方差。关键发现是 OPD 的 value function 有闭合解：per-token 的 student-teacher reverse KL divergence，可直接从 forward pass 中获得，无需额外 critic 或推理开销。所提方法 vOPD 在保留单样本估计量无偏性的同时，通过减去该 baseline 降低梯度方差，并用 top-k 近似进一步降低计算成本；在数学与科学推理 benchmark 上一致优于 vanilla OPD，且匹配计算代价最高的 full-vocabulary baseline。

- 🎯 方法：将 OPD 的梯度估计纳入 RL policy-gradient 框架，用 per-token reverse KL 作为 control variate baseline 降低方差
- 💡 创新：OPD value function 有闭合解，直接来自 forward pass，无需额外 critic 或额外推理
- 🔧 工程：top-k 近似 baseline 进一步降低计算开销，且不损失性能
- 📊 结果：在数学和科学推理 benchmark 上优于 vanilla OPD，匹配 full-vocabulary baseline 效果
