#### 摘要
On-Policy Distillation (OPD) 因单样本 Monte Carlo 估计器的梯度方差过高而训练不稳定。本文将 OPD 形式化为 policy-gradient RL，并引入 control variate baseline（即 value function）来降低方差。关键发现是 OPD 的 value function 有闭合解：即 student 与 teacher 之间的 per-token negative reverse KL divergence，可直接从已有 forward pass 中获取，无需额外 critic 或推理开销。vOPD 保留了轻量的单样本估计器，将该 value function 作为 detached baseline 减去，保持梯度无偏的同时降低方差；在数学和科学推理 benchmark 上，vOPD 持续优于 vanilla OPD，并与开销最大的 full-vocabulary baseline 性能持平。

- 🎯 将 OPD 梯度方差问题转化为 RL variance reduction 问题，引入 control variate baseline 稳定训练
- 💡 OPD value function 闭合解为 per-token negative reverse KL，无需额外 critic 或推理即可获得
- 📊 在数学和科学推理 benchmark 上超越 vanilla OPD，匹配 full-vocabulary baseline 性能
