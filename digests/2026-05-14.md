#### 摘要
Orthrus 在冻结的 autoregressive LLM 上挂载一个轻量可训练的 diffusion head，构成 dual-view 架构：AR view 负责 prefill 并构建高保真 KV cache，diffusion view 复用同一 KV cache 进行并行 token 生成。两个 view 通过 exact consensus 机制对齐输出，保证 lossless inference。相比纯 diffusion LM 的性能退化和高训练成本，该方法以 O(1) 额外 cache 开销和极少新增参数实现最高 7.8x 解码加速。

- 🎯 AR head 做 prefill，diffusion head 并行生成共享 KV cache
- 💡 exact consensus 机制保证 lossless inference
- 📊 最高 7.8x 加速，O(1) cache 开销
- 🔧 轻量模块可嵌入现有 Transformer，冻结原 LLM
