#### 摘要
SANA-Video 针对高分辨率长视频生成的效率瓶颈，提出两项核心设计：以 linear attention 替代 vanilla attention 构建 Linear DiT，以及基于 linear attention 累积特性设计的 constant-memory KV cache，用于 block-wise 自回归长视频生成，彻底消除传统 KV cache 的内存随序列增长问题。该模型可在 RTX 5090 GPU 上以 NVFP4 精度部署，生成 5 秒 720p 视频从 71s 加速至 29s（2.4x）。在训练开销仅为 MovieGen 1% 的前提下（64 H100 GPU 训练 12 天），与 Wan 2.1-1.3B、SkyReel-V2-1.3B 等同量级模型相比，延迟降低 16x，且保持竞争性生成质量。

- 🎯 Constant-memory KV cache 基于 linear attention 累积特性，固定内存开销支持分钟级长视频生成
- 📊 与 Wan 2.1-1.3B / SkyReel-V2-1.3B 相比，测量延迟低 16x
- 📊 RTX 5090 + NVFP4 部署，5 秒 720p 视频推理从 71s 降至 29s（2.4x speedup）
