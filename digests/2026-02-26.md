**Diffusion 压缩** · topic 4/5 · practicality 4/5 · composite 8/10 · 扩散模型压缩工作，结合线性注意力架构设计与NVFP4量化部署；推理加速明显（2.4x），但缺乏传统精度基准（MMLU等）评估。定性质量对标竞品达到可竞争水平。算法设计简洁（线性注意力替代标准注意力），内存高效。虽然主要贡献是架构创新而非显式量化方法，但NVFP4部署案例与常数内存KV cache设计符合压缩范畴。

#### 摘要
SANA-Video 针对高分辨率长视频生成的效率瓶颈，提出两项核心设计：以 linear attention 替代 vanilla attention 的 Linear DiT，以及基于 linear attention 累积性质的 constant-memory KV cache Block Linear Attention，使模型在固定显存开销下实现分钟级视频的自回归生成。训练成本仅为 64 张 H100 GPU 上 12 天，约为 MovieGen 的 1%。与同量级模型（Wan 2.1-1.3B、SkyReel-V2-1.3B）相比，生成延迟降低 16×；在 RTX 5090 上以 NVFP4 精度部署，5 秒 720p 视频生成从 71s 加速至 29s（2.4× speedup）。

- 🎯 Constant-memory KV cache 基于 linear attention 累积性质，以固定显存实现分钟级长视频 block-wise 自回归生成
