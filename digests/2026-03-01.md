#### 摘要
SANA-Video 针对高分辨率长视频生成的效率瓶颈，提出两项核心设计：用 linear attention 替代 vanilla attention 以降低大量 token 的计算开销，并设计基于 linear attention 累积性质的 constant-memory KV cache block-wise autoregressive 方案，在固定显存代价下为 Linear DiT 提供全局上下文。模型在 64 张 H100 GPU 上训练 12 天（约为 MovieGen 训练成本的 1%），可生成最高 720x1280、分钟级时长视频。与 Wan 2.1-1.3B、SkyReel-V2-1.3B 等同规模模型相比性能相当，实测延迟快 16x；在 RTX 5090 以 NVFP4 精度部署时，生成 5 秒 720p 视频从 71s 降至 29s（2.4x 加速）。

- 🎯 linear attention 替代 vanilla attention，配合 constant-memory KV cache 实现分钟级长视频生成
- 📊 训练成本仅 12 天 × 64 H100，约为 MovieGen 的 1%
- 📊 与 Wan 2.1-1.3B / SkyReel-V2-1.3B 性能相当，实测延迟快 16x
- 📊 RTX 5090 + NVFP4 部署，5 秒 720p 视频推理从 71s 加速至 29s（2.4x）
