#### 摘要
SANA-Video 是一个小型扩散模型，目标是在 RTX 5090 GPU 上高效生成最高 720x1280 分辨率、分钟级时长的视频。核心方法有两点：用 linear attention 替代 vanilla attention 构建 Linear DiT，以及为 Block Linear Attention 设计基于线性 attention 累积性质的 constant-memory KV cache，以固定显存代价提供全局上下文。训练仅需 64 H100 GPU 跑 12 天（约为 MovieGen 成本的 1%），与同规模 SOTA 模型相比延迟降低 16x，并通过 NVFP4 量化将 5 秒 720p 视频的生成时间从 71s 压缩至 29s（2.4x 加速）。

- 🎯 linear attention 构建 Linear DiT + constant-memory KV cache，支持分钟级长视频生成
- 📊 与 Wan 2.1-1.3B、SkyReel-V2-1.3B 竞争力相当，实测延迟降低 16x
- 📊 NVFP4 量化部署于 RTX 5090，5秒 720p 视频生成从 71s→29s（2.4x speedup）
- 🔧 训练成本仅 64 H100 GPU × 12 天，约为 MovieGen 的 1%
