#### 摘要
SANA-Video 针对高分辨率长视频生成的效率瓶颈，提出两项核心设计：用 linear attention 替代 vanilla attention 以降低大 token 数下的计算复杂度，并设计基于 linear attention 累积性质的 constant-memory KV cache，实现 block-wise autoregressive 长视频生成而无需传统 KV cache 的线性显存增长。训练成本仅为 MovieGen 的 1%（64 张 H100 GPU 训练 12 天），生成速度比同规模模型（Wan 2.1-1.3B、SkyReel-V2-1.3B）快 16x，在 RTX 5090 上以 NVFP4 精度推理 5 秒 720p 视频从 71s 降至 29s（2.4x 加速）。

- 🎯 Block Linear Attention 的 constant-memory KV cache，以固定显存代价提供全局上下文，支持分钟级长视频生成
- 📊 与 Wan 2.1-1.3B、SkyReel-V2-1.3B 同等质量下，推理延迟快 16x
- 📊 RTX 5090 + NVFP4 精度推理 5 秒 720p 视频：71s → 29s（2.4x 加速）
- 🔧 训练成本约为 MovieGen 的 1%（64 H100 GPU × 12 天）
