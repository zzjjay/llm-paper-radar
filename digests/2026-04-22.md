#### 摘要
SANA-Video 是一个面向高效视频生成的小型扩散模型，支持最高 720x1280 分辨率和分钟级时长视频生成。核心设计包括：Linear DiT（以 linear attention 替代 vanilla attention 以降低大规模 token 处理的计算开销）和 Constant-Memory KV cache for Block Linear Attention（利用 linear attention 的累积性质设计 block-wise 自回归生成，以固定内存代价提供全局上下文）。训练成本仅为 12 天 × 64 H100 GPUs（约为 MovieGen 的 1%），在同规模模型对比中推理速度快 16x，在 RTX 5090 上部署 NVFP4 精度后生成 5 秒 720p 视频延迟从 71s 降至 29s。

- 🎯 Linear DiT + Constant-Memory KV cache 实现 block-wise 自回归长视频生成，内存开销固定
- 📊 与 Wan 2.1-1.3B、SkyReel-V2-1.3B 相当质量，推理速度快 16x
- 📊 NVFP4 部署在 RTX 5090 上，5 秒 720p 视频生成从 71s 加速至 29s（2.4x speedup）
- 🔧 训练成本仅为 12 天 × 64 H100 GPUs，约为 MovieGen 的 1%
