**Diffusion 压缩** · topic 4/5 · practicality 4/5 · composite 8/10 · 扩散模型压缩工作，采用线性注意力机制降低计算复杂度，引入常数内存KV缓存设计实现长视频高效生成，NVFP4量化实现2.4倍推断加速。虽未明确报告定量精度指标（如FID），但通过与SOTA小模型对比和明显的延迟改进展示实用价值。缺少标准视频生成精度基准评估（FID/LPIPS）、NVFP4标定成本细节影响评分。

#### 摘要
SANA-Video 针对高分辨率长视频生成的效率问题，提出两项核心设计：用 linear attention 替代 vanilla attention 以降低大量 token 处理的计算开销，以及基于 linear attention 累积性质设计的 constant-memory KV cache 的 block-wise autoregressive 方案，实现固定显存代价下的全局上下文建模。模型可生成最高 720x1280、分钟级时长视频，训练仅需 64 张 H100 GPU 跑 12 天（约为 MovieGen 训练成本的 1%）。在同等小模型对比中，相比 Wan 2.1-1.3B 和 SkyReel-V2-1.3B 速度快 16x，部署于 RTX 5090 并以 NVFP4 精度推理时，生成 5 秒 720p 视频延迟从 71s 降至 29s（2.4x 加速）。

- 🎯 用 linear attention 构建 Linear DiT，配合 constant-memory KV cache 实现分钟级长视频 block-wise autoregressive 生成
- 📊 训练成本仅 64 H100 × 12 天，约为 MovieGen 的 1%
- 📊 推理速度比 Wan 2.1-1.3B 和 SkyReel-V2-1.3B 快 16x；RTX 5090 上 NVFP4 推理实现 2.4x 加速（71s→29s）
- 💡 Constant-memory KV cache 利用 linear attention 的累积性质，消除传统 KV cache 随长度线性增长的显存需求
