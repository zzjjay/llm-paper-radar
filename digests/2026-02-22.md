#### 摘要
SANA-Video 针对高分辨率长视频生成的效率瓶颈，提出了两项核心设计：用 linear attention 替换 vanilla attention 以降低大 token 数下的计算开销，以及基于 linear attention 累积特性设计 constant-memory KV cache 的 block-wise 自回归方案，使显存占用固定且支持分钟级视频生成。模型可在 RTX 5090 上以 NVFP4 精度部署，生成 5 秒 720p 视频从 71s 降至 29s（2.4x 加速）。与同量级的 Wan 2.1-1.3B 和 SkyReel-V2-1.3B 相比，SANA-Video 在 competitive 质量下实现 16x 延迟加速，而训练成本仅为 MovieGen 的 1%（64 H100 GPUs 训练 12 天）。

- 🎯 Linear DiT + constant-memory KV cache 实现固定显存开销的分钟级长视频自回归生成
