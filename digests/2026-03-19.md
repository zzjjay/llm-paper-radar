#### 摘要
LLM 在仅训练 next-token prediction 的情况下隐含 multi-token prediction 能力。该工作提出一种无需训练的 MTP 方法：在推理时从 embedding space 动态采样 mask token，构造 speculative token tree，通过 top-K candidate 采样与轻量剪枝保留高概率续写路径，再并行验证候选 token 以实现无损生成。方法在 LLaMA3 上将 acceptance length 提升约 12%，在 Qwen3 上提升 8–12%，throughput 提升最高达 15–19%，且无需修改模型权重或引入 auxiliary draft model。

- 🎯 方法：推理时用 embedding space 中的 mask token 探测 LLM，无需训练或辅助模型即可实现并行 multi-token prediction
- 📊 结果：LLaMA3 acceptance length 提升约 12%，Qwen3 提升 8–12%，throughput 最高提升 15–19%
- 💡 创新：理论与实验表明 decoder 层自然将 mask-token 表示与 next-token 状态对齐，支撑多步预测
- 🔧 工程：构造 speculative token tree + 轻量剪枝策略，并行验证保证生成无损
