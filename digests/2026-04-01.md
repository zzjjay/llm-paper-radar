#### 摘要
SANA-Video 针对高分辨率长视频生成的效率瓶颈，提出两项核心设计：以 linear attention 替代 vanilla attention 构建 Linear DiT，以及基于 linear attention 累积性质的 constant-memory KV cache，支持逐块自回归生成而不随视频长度增加显存开销。训练成本压缩至 64 张 H100 GPU 上 12 天（仅为 MovieGen 的 1%），可生成 720x1280 分辨率、分钟级时长视频。在小模型对比中，SANA-Video 相对 Wan 2.1-1.3B 和 SkyReel-V2-1.3B 实现 16x 延迟加速，并可在 RTX 5090 GPU 上以 NVFP4 精度将 5 秒 720p 视频生成延迟从 71s 降至 29s（2.4x 加速）。

- 🎯 Constant-memory KV cache 基于 linear attention 累积性质，以固定显存实现分钟级长视频块状自回归生成
- 📊 对比 Wan 2.1-1.3B / SkyReel-V2-1.3B，实测延迟快 16x，质量持平
- 📊 RTX 5090 + NVFP4 推理，5 秒 720p 视频生成从 71s 降至 29s（2.4x speedup）
- 🔧 训练仅需 64 × H100 × 12 天，约为 MovieGen 训练成本的 1%
