#### 摘要
SANA-Video 是一个高效视频生成扩散模型，支持最高 720x1280 分辨率和分钟级时长的视频生成。核心设计包含两点：用 linear attention 替代 vanilla attention 以降低大量 token 场景下的计算开销；设计基于 block-wise autoregressive 的 Constant-Memory KV cache，利用 linear attention 的累积性质维护固定内存的全局上下文，彻底消除传统 KV cache 的线性内存增长问题。训练成本仅为 64 张 H100 GPU 训练 12 天（约为 MovieGen 的 1%），在同等小模型对比中延迟比 Wan 2.1-1.3B 和 SkyReel-V2-1.3B 快 16x，并支持在 RTX 5090 上以 NVFP4 精度部署，生成 5 秒 720p 视频从 71s 加速至 29s。

- 🎯 Constant-Memory KV cache：基于 linear attention 累积性质设计 block-wise autoregressive 状态，固定内存实现分钟级长视频生成
- 📊 相比 Wan 2.1-1.3B 和 SkyReel-V2-1.3B，推理延迟快 16x，训练成本仅为 MovieGen 的 1%（64×H100，12 天）
- 📊 RTX 5090 上 NVFP4 精度部署，5 秒 720p 视频生成从 71s 降至 29s（2.4x speedup）
- 💡 用 Linear DiT 替代 vanilla attention，适配视频生成中 token 数量极大的场景
