#### 摘要
SANA-Video 针对长视频生成中 token 数量庞大导致的计算和显存瓶颈，提出 Linear DiT 架构，以线性 attention 替代 vanilla attention 降低计算复杂度。核心创新是基于线性 attention 累积性质设计的 Constant-Memory KV cache Block Linear Attention，以固定显存开销提供全局上下文，支持分钟级长视频生成。训练仅需 64 台 H100 GPU 运行 12 天（约为 MovieGen 的 1%），在与 Wan 2.1-1.3B 等同规模模型对比中实现 16x 延迟加速，部署于 RTX 5090 配合 NVFP4 精度后 720p 5 秒视频生成从 71s 降至 29s。

- 🎯 Linear DiT + Constant-Memory KV cache，以固定显存实现分钟级 720x1280 视频生成
- 📊 与 Wan 2.1-1.3B / SkyReel-V2-1.3B 同规模模型相比，实测延迟快 16x
- 📊 RTX 5090 + NVFP4 部署：720p 5 秒视频生成从 71s 加速至 29s（2.4x speedup）
- 🔧 训练成本 64×H100 GPU × 12 天，约为 MovieGen 的 1%
