#### 摘要
SANA-Video 针对高分辨率长视频生成的效率瓶颈，提出两项核心设计：以 linear attention 替代 vanilla attention 构建 Linear DiT，以及基于 linear attention 累积特性设计 constant-memory KV cache 的 block-wise autoregressive 生成框架，从而在固定显存开销下实现分钟级视频生成。训练成本仅为 MovieGen 的 1%（64 张 H100 GPU 训练 12 天），在与 Wan 2.1-1.3B、SkyReel-V2-1.3B 等同量级模型对比中实现 16x latency 加速。在 RTX 5090 上以 NVFP4 精度部署后，生成 5 秒 720p 视频的推理时间从 71s 降至 29s（2.4x 加速）。

- 🎯 用 linear attention 构建 Linear DiT，constant-memory KV cache 支持 block-wise autoregressive 长视频生成
- 📊 与 Wan 2.1-1.3B、SkyReel-V2-1.3B 相比，latency 快 16x，训练成本仅为 MovieGen 的 1%
- 📊 RTX 5090 上 NVFP4 精度推理，5 秒 720p 视频从 71s 降至 29s，2.4x 加速
- 🔧 可在单张 RTX 5090 GPU 上部署，支持 720x1280 分辨率及分钟级时长视频生成
