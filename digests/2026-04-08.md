#### 摘要
SANA-Video 针对高分辨率长视频生成中 attention 计算与 KV cache 内存开销过大的问题，提出两项核心设计：用 linear attention 替代 vanilla attention 以降低 token 数量增长带来的计算代价，并基于 linear attention 的累积性质设计 constant-memory KV cache 的 block-wise autoregressive 方案，实现固定内存代价的全局上下文建模。模型可生成 720×1280 分辨率、分钟级时长视频，训练仅需 64 张 H100 GPU 跑 12 天（约为 MovieGen 训练成本的 1%）。在 RTX 5090 上部署 NVFP4 精度推理，生成 5 秒 720p 视频从 71s 加速至 29s（2.4× speedup），且在质量上与 Wan 2.1-1.3B、SkyReel-V2-1.3B 等同量级模型相当，推理延迟快 16×。

- 🎯 Linear DiT + constant-memory block-wise KV cache，实现分钟级长视频在固定显存下的自回归生成

- 📊 RTX 5090 上 NVFP4 推理 5s 720p 视频：71s → 29s（2.4× speedup），vs 同级模型快 16×
