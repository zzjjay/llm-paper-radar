#### Summary
LongLive-2.0 is an end-to-end NVFP4-based parallel infrastructure addressing speed and memory bottlenecks across long video generation training and inference. Training uses Balanced SP, which pairs clean-history and noisy-target temporal chunks per rank to co-design teacher-forcing with SP execution and SP-aware chunked VAE encoding. It directly tunes a diffusion model into a multi-shot interactive AR diffusion model, bypassing the ODE initialization plus DMD distillation used in Self-Forcing variants, and can be converted to 2-4 step real-time generation via standalone LoRA. Inference on Blackwell GPUs uses W4A4 NVFP4, NVFP4-quantized KV cache, and asynchronous streaming VAE decoding; non-Blackwell GPUs rely on SP inference. Reported gains: 2.15x training speedup, 1.84x inference speedup, and 45.7 FPS for LongLive-2.0-5B.

- 🎯 Balanced SP pairs clean/noisy chunks per rank with NVFP4 GEMM training
- 💡 Directly tunes diffusion into AR diffusion, skipping ODE init and DMD distillation
- 🔧 W4A4 NVFP4 inference + NVFP4 KV cache + async streaming VAE decoding
- 📊 2.15x training and 1.84x inference speedup; 45.7 FPS at 5B

#### Summary
Existing LLM serving systems waste large amounts of KV cache memory through fragmentation and redundant duplication, limiting batch size and throughput. PagedAttention adapts classical OS virtual-memory and paging techniques to KV cache management, enabling near-zero memory waste and flexible KV cache sharing within and across requests. Built on top, vLLM delivers 2-4× higher throughput than FasterTransformer and Orca at the same latency, with larger gains on longer sequences, larger models, and more complex decoding algorithms.

- 💡 OS-style paging applied to KV cache eliminates fragmentation
- 🎯 Enables flexible KV cache sharing within and across requests
- 📊 2-4× throughput over FasterTransformer/Orca at equal latency
- 🔧 Gains grow with longer sequences, larger models, complex decoding
