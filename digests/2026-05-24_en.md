#### Summary
LongLive-2.0 builds an NVFP4-based parallel infrastructure across training and inference for long video generation. Training uses Balanced SP, which pairs clean-history and noisy-target temporal chunks on each rank and co-designs teacher-forcing layout with SP-aware chunked VAE encoding, combined with NVFP4 to cut memory and accelerate GEMM. Unlike Self-Forcing variants relying on ODE init plus DMD, it directly tunes a diffusion model into a multi-shot interactive AR diffusion model, convertible to 2-step real-time generation via LoRA. Inference on Blackwell enables W4A4 NVFP4 with NVFP4 KV cache and asynchronous streaming VAE decoding; non-Blackwell uses SP inference. Reports 2.15x training and 1.84x inference speedup, with LongLive-2.0-5B hitting 45.7 FPS.

- 🎯 Balanced SP pairs clean-history and noisy-target chunks per rank with SP-aware chunked VAE encoding
- 🔧 W4A4 NVFP4 inference, NVFP4 KV cache, asynchronous streaming VAE decoding
- 📊 2.15x training and 1.84x inference speedup; 5B model reaches 45.7 FPS
- 💡 First NVFP4 training+inference system for long video generation

#### Summary
Addresses KV cache memory waste from fragmentation and redundant duplication that limits batch size in LLM serving. Proposes PagedAttention, an attention algorithm inspired by OS virtual memory and paging, and builds vLLM on top, achieving near-zero KV cache waste plus flexible sharing within and across requests. Evaluations show 2-4× throughput improvement over FasterTransformer and Orca at the same latency, with larger gains for longer sequences, bigger models, and complex decoding.
