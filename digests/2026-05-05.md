#### 🧭 为什么推送这篇
**Diffusion 压缩** · topic 4/5 · practicality 4/5 · composite 8/10 · 扩散模型压缩核心贡献：线性注意力替代标准注意力，常数内存 KV cache 设计用于长视频生成，NVFP4 量化实现 2.4x 推理加速。强有力的推理性能对标现代小型扩散模型。但缺乏传统图像质量指标（FID/IS）评估，降低话题相关性至 4；实用性 4 分因为算法简洁、推理加速明显、部署于消费级 GPU。

#### 摘要
SANA-Video 针对高分辨率长视频生成的效率瓶颈，提出两项核心设计：用 linear attention 替换 vanilla attention 构建 Linear DiT，以及基于 linear attention 累积性质的 constant-memory KV cache，以固定显存支持分块自回归长视频生成。训练成本压缩至 64 张 H100 GPU 跑 12 天（约为 MovieGen 的 1%），可在 RTX 5090 上以 NVFP4 精度部署。与同量级模型 Wan 2.1-1.3B、SkyReel-V2-1.3B 相比性能持平，推理延迟低 16x；NVFP4 量化将 720p 5 秒视频生成时间从 71s 压缩至 29s（2.4x 加速）。

- 🎯 Constant-memory KV cache：基于 linear attention 累积特性，以固定显存实现分块自回归长视频生成，消除传统 KV cache 随长度线性增长问题
- 📊 与 Wan 2.1-1.3B、SkyReel-V2-1.3B 性能相当，推理延迟快 16x
- 📊 RTX 5090 上 NVFP4 量化将 720p 5s 视频生成从 71s 降至 29s，实现 2.4x 加速
- 🔧 训练成本仅 64×H100 GPU 12 天，约为 MovieGen 的 1%
