#### 摘要
SANA-Video 针对视频生成中 token 数量庞大导致的计算与内存瓶颈，提出两项核心设计：用 linear attention 替代 vanilla attention 的 Linear DiT，以及基于 linear attention 累积性质的 constant-memory KV cache block-wise autoregressive 生成方案，使长视频生成的内存开销固定。模型可生成 720x1280 分辨率、分钟级时长视频，训练成本仅需 64 块 H100 GPU 训练 12 天（约为 MovieGen 的 1%）。在 RTX 5090 上以 NVFP4 精度部署时，生成 5 秒 720p 视频从 71s 降至 29s（2.4x 加速），与 Wan 2.1-1.3B、SkyReel-V2-1.3B 等同量级模型相比延迟降低 16x。

- 🎯 Linear DiT + constant-memory KV cache 实现固定内存开销的分钟级长视频 block-wise autoregressive 生成
- 📊 训练成本仅 64 H100 × 12 天，约为 MovieGen 的 1%
- 📊 NVFP4 部署在 RTX 5090 上，5 秒 720p 视频推理从 71s→29s（2.4x 加速）
- 📊 与 Wan 2.1-1.3B、SkyReel-V2-1.3B 性能相当，测量延迟快 16x
