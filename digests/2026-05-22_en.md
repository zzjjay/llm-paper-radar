#### Summary
Addresses KV cache memory fragmentation and redundant duplication that limit batch size in LLM serving. Proposes PagedAttention, an attention algorithm inspired by OS virtual memory and paging, managing KV cache in blocks. Built on top of it, vLLM achieves near-zero KV cache waste and flexible sharing within and across requests. At matched latency, vLLM improves throughput of popular LLMs by 2-4× over FasterTransformer and Orca, with larger gains on longer sequences, bigger models, and more complex decoding algorithms.

- 💡 OS-style virtual memory/paging applied to KV cache
- 📊 2-4× throughput vs FasterTransformer/Orca at same latency
- 🎯 Near-zero KV cache waste + intra/inter-request sharing
