#### 摘要
本文在 lossy（GPTQ/AWQ）与严格无损之间提出 statistically-lossless 量化框架，定义三种 losslessness：task-lossless（zero-shot 精度落在采样方差内）、distribution-lossless（用 Expected Acceptance Rate (EAR) 衡量 next-token 分布一致性，EAR≥0.99 表示 99% token 一致）、以及证明 symmetric quantization 相对 asymmetric 会将噪声方差放大 γ² 的 γ² variance law。基于此，作者提出 SLQ：layer-wise 非均匀、asymmetric quantization 加宽 bitwidth search。结果在 task-lossless 上低至 3.3 bits/param，distribution-lossless 平均 5–6 bits/param，配合优化 kernel 相对 FP16 实现 1.7–3.6× 推理加速。

- 💡 提出 EAR 度量分布无损：EAR≥0.99 即 99% token 一致
- 💡 证明 γ² variance law：symmetric 比 asymmetric 噪声方差大 γ²
- 📊 task-lossless 低至 3.3 bits/param，dist-lossless 5–6 bits
- 📊 相对 FP16 推理加速 1.7–3.6×
