#### 摘要
SANA-Video 是一个小型 diffusion model，目标是在 RTX 5090 GPU 上高效生成最高 720x1280 分辨率、分钟级时长的视频。核心技术有两点：一是用 linear attention 替代 vanilla attention 以降低 token 数爆炸带来的计算开销；二是为 Block Linear Attention 设计 constant-memory KV cache，利用 linear attention 的累积性质实现固定显存的全局上下文，支持分钟级长视频生成。训练成本仅为 64 张 H100 GPU 训练 12 天（MovieGen 的 1%），推理速度比 Wan 2.1-1.3B 和 SkyReel-V2-1.3B 快 16x；在 RTX 5090 上以 NVFP4 精度部署后，生成 5 秒 720p 视频从 71s 降至 29s（2.4x 加速）。

- 🎯 Block Linear Attention 配合 constant-memory KV cache，固定显存代价实现分钟级长视频生成
- 📊 与同规模模型相比推理延迟降低 16x，NVFP4 部署在 RTX 5090 上实现 2.4x 加速（71s→29s）
- 📊 训练成本仅 64×H100 GPU × 12 天，约为 MovieGen 的 1%
