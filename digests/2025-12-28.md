#### 🧭 为什么推送这篇
**QAT（量化感知训练）** · topic 4/5 · practicality 3/5 · composite 7/10 · 本文提出MoR（Mixture-of-Representations），一种在FP8与BF16之间动态切换的混合精度训练框架，支持per-tensor和sub-tensor粒度。方法具有一定创新性，动态分析张量数值属性来选择表示格式，有望改善低精度训练的鲁棒性。但存在几个局限：(1)属于'preliminary study'，评估结果不够完整，缺乏标准下游基准（MMLU、GSM8K等）的系统评测；(2)最大测试模型规模未知；(3)本质是训练时量化（QAT/混合精度训练），需要完整训练流程，部署成本较高；(4)算法复杂度中等，动态张量属性分析需要额外计算开销。综合来看，topic_relevance=4（核心量化方向但评估薄弱），practicality=3（有合理的部署路径，但需要完整训练且规模验证不足）。

#### 摘要
针对混合精度训练中如何为不同 tensor 选择数值格式的问题，提出 Mixture-of-Representations (MoR) 框架，在 per-tensor 和 sub-tensor 粒度上根据张量数值特性动态在 FP8 和 BF16 之间切换。该方法无需细粒度 partition 即可保持模型质量，初步实验显示 98.38% 的 tensor 可量化到 FP8 并达到 SOTA 精度。作者认为该方法亦可与 NVFP4 等更低精度格式结合使用。

- 💡 按张量数值属性动态选 FP8 或 BF16
- 📊 98.38% tensor 用 FP8 仍达 SOTA
- 🎯 支持 per-tensor 与 sub-tensor 双粒度切换
- 🔧 可叠加 NVFP4 等更低精度方案

#### 🧭 为什么推送这篇
**Pruning / 蒸馏** · topic 4/5 · practicality 3/5 · composite 7/10 · FANG是一个post-training结构化剪枝框架，通过功能感知神经元分组缓解calibration偏差，与FLAP和OBC结合实现SOTA，在30%和40%稀疏度下超越基线1.5%–8.5%。方法有实际部署潜力（post-training，无需梯度），且结构化剪枝结果可直接加速推理。但摘要未注明最大测试模型尺寸（可能是Llama系列但未确认），且无端到端延迟/吞吐量数据。评分适中：方法是在现有剪枝基础上的改进，算法复杂度中等（神经元分组+重要性加权+自适应稀疏分配），缺乏详细的GPU加速实测数据使practicality略受限。

#### 摘要
针对 LLM 后训练结构化剪枝中 few-shot calibration set 难以反映预训练分布、导致下游泛化差的问题，提出 Function-Aware Neuron Grouping (FANG)。该方法按神经元处理的语义上下文类型将其分组，组内独立剪枝并用与该功能角色强相关的 token 加权重要性估计，同时保留跨多种上下文类型贡献的神经元，并根据 block 功能复杂度自适应分配 sparsity。在 30% 和 40% sparsity 下，与 FLAP、OBC 结合后平均精度分别超出原方法 1.5%–8.5%，达到 SOTA。

- 💡 按神经元功能角色分组，组内独立剪枝
- 🎯 功能相关 token 加权重要性估计，保留多功能神经元
- 🔧 按 block 功能复杂度自适应分配 sparsity
- 📊 30%/40% sparsity 下比 FLAP/OBC 高 1.5%–8.5%
