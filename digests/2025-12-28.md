### 2. Improving Generalization in LLM Structured Pruning via Function-Aware Neuron Grouping (7/10)

👥 Tao Yu, Yongqi An, Kuan Zhu... · 🏷 cs.CL

🧪 pruning_distill · 结构化剪枝，函数感知神经元分组 · cal: post-training，需少量校准数据，成本较低 · perf: 结构化剪枝可直接加速，但无具体端到端延迟数据

#### 🧭 为什么推送这篇
**Pruning / 蒸馏** · topic 4/5 · practicality 3/5 · composite 7/10 · 本文提出FANG，一种基于功能感知神经元分组的post-training结构化剪枝框架，针对LLM在few-shot校准集分布偏移时泛化性差的问题。方法将神经元按语义上下文类型分组，分别剪枝并加权重要性估计，同时自适应分配各块稀疏度。结合FLAP和OBC两个代表性剪枝基线实现SOTA，在30%/40%稀疏度下有明显精度提升。方法属于有实际部署价值的结构化剪枝范畴，算法思路清晰，无需梯度更新，属post-training范式。但摘要未明确说明最大测试模型规模，无具体推理延迟/吞吐数据，实用性有一定不确定性，故综合评分适中。

#### 摘要
针对 post-training structured pruning 中 few-shot calibration set 难以反映预训练分布、导致下游泛化差的问题，提出 Function-Aware Neuron Grouping (FANG)。该方法按神经元处理的语义上下文类型将其分组，组内独立做重要性估计并对与该功能相关的 token 加权，同时保留跨多种上下文都活跃的神经元，并依据 block 的功能复杂度自适应分配 sparsity。结合 FLAP 与 OBC 后在 30% 与 40% sparsity 下平均准确率超出两者 1.5%–8.5%，达到 SOTA。

- 🎯 按 semantic context 类型对神经元分组并独立剪枝
- 💡 token 按神经元功能角色加权，保留跨上下文神经元
- 🔧 依 block 功能复杂度自适应分配 sparsity
- 📊 30%/40% sparsity 下平均准确率超 FLAP/OBC 1.5%–8.5%
