- 🎯 方法：compressed KV cache 做 draft，full KV cache（驻 CPU/远端）做 verification，实现无损 LLM 推理
- 💡 创新：HBM-bound 的 compressed-KV decoding 与 PCIe/network-bound 的 full-KV swap 并行，消除 swap 开销
- 📊 结果：相比 full-KV inference，吞吐量最高提升 4×，输出与 full-KV 完全一致
- 🔧 工程：统一 compressor 接口支持 token dropping 和 quantization 系列方法，兼容 long-context 与 remote prefix caching
