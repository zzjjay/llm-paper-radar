#### 摘要
SANA-Video 是一个小型 diffusion 模型，目标是在单张消费级 GPU（RTX 5090）上高效生成最高 720×1280 分辨率、分钟级时长的视频。核心设计有两点：以 linear attention 替代 vanilla attention 构建 Linear DiT，以及基于 linear attention 累积性质设计 constant-memory KV cache 的 block-wise autoregressive 生成机制，使长视频生成的显存开销固定。训练仅需 64 张 H100 GPU 跑 12 天（约为 MovieGen 的 1%），在与 Wan 2.1-1.3B 和 SkyReel-V2-1.3B 同级别模型的对比中延迟快 16×，部署 NVFP4 精度后生成 5 秒 720p 视频从 71s 降至 29s。

- 🎯 Constant-memory KV cache：利用 linear attention 累积性质实现 block-wise autoregressive，显存固定支持分钟级视频生成
- 📊 与 Wan 2.1-1.3B / SkyReel-V2-1.3B 相比，延迟快 16×，质量持平
- 📊 NVFP4 精度部署在 RTX 5090 上，5s 720p 视频生成从 71s 降至 29s（2.4× speedup）
- 🔧 训练成本仅 64×H100 GPU × 12 天，约为 MovieGen 的 1%
