#### 摘要
针对 2-bit KV cache 量化在长上下文推理中精度下降的问题，提出 Kitty 的算法-系统协同设计。算法上采用 Dynamic Channel-wise Precision Boost，按敏感度对 Key cache 通道排序，仅将少量通道保留为 4-bit，其余压至 2-bit。系统上将混合精度 Key page 分解为两个统一 2-bit 张量，配合 page-centric KV layout 与 Triton 兼容的 dequantization kernel，保持访存 coalescing 并避免 warp divergence。在 Qwen3、LLaMA3 上 7 个任务实现近 8x KV 显存压缩、batch 扩大 8x、吞吐提升 2.1x-4.1x，精度损失可忽略。
