#### 🧭 为什么推送这篇
**KV cache 压缩** · topic 5/5 · practicality 5/5 · composite 10/10 · Kitty 是一个面向 KV cache 的 2-bit 混合精度量化方案，核心创新是动态 channel-wise Precision Boost：对 Key cache 按 channel 敏感度排序，仅保留小部分高精度 channel，其余降至 2-bit，从而在接近 2-bit 内存开销下保持精度。系统侧通过将混合精度 page 分解为两个统一 2-bit tensor，提供 page-centric layout、Triton-compatible 反量化 kernel 和轻量 runtime pipeline，避免 scattered reads 和 hard-coded masks，实现 coalesced 内存访问。在 Qwen3 和 LLaMA3 上覆盖七类任务，报告 2.1x-4.1x 吞吐提升和 8x KV 内存压缩。代码完全开源，工程路径清晰，practicality 极高。

#### 摘要
针对 2-bit KV cache 量化在长上下文推理上掉点严重的问题，Kitty 提出算法-系统协同设计的混合精度方案。算法上引入 Dynamic Channel-wise Precision Boost：按敏感度对 Key cache 通道排序，仅保留少量高敏感通道为 4-bit，其余压至 2-bit。系统上将每个混合精度 Key page 分解为两个统一 2-bit 张量，配合 page-centric KV 布局、Triton 兼容的 dequantization kernel 和轻量运行时流水线，避免分散读取和分支发散。在 Qwen3、LLaMA3 上跨 7 个任务实现近 8x KV 显存压缩，精度损失可忽略，同等显存预算下 batch 提升 8x、吞吐提升 2.1x-4.1x。
