#### 摘要
SANA-Video 是一个小型扩散模型，目标是在有限计算资源下高效生成最高 720x1280 分辨率、分钟级时长的视频。核心设计包含两点：(1) Linear DiT，用 linear attention 替代 vanilla attention 以降低大量 token 处理时的计算复杂度；(2) Constant-Memory KV cache for Block Linear Attention，利用 linear attention 的累积特性设计 block-wise 自回归方案，以固定内存代价提供全局上下文，消除传统 KV cache 的内存线性增长问题。训练成本仅需 64 块 H100 GPU 运行 12 天（约为 MovieGen 的 1%），生成速度比同量级模型（Wan 2.1-1.3B、SkyReel-V2-1.3B）快 16x，在 RTX 5090 上以 NVFP4 精度推理 5 秒 720p 视频从 71s 加速至 29s（2.4x speedup）。

- 🎯 用 linear attention 构建 Linear DiT，配合 constant-memory block-wise KV cache 实现分钟级长视频生成
- 📊 训练成本仅为 MovieGen 的 1%（64 H100 GPU × 12 天），推理速度比 Wan 2.1-1.3B / SkyReel-V2-1.3B 快 16x
- 📊 RTX 5090 上以 NVFP4 精度推理，5 秒 720p 视频生成从 71s 降至 29s（2.4x speedup）
- 💡 linear attention 的累积特性使 KV cache 内存固定，无需随视频长度线性增长
