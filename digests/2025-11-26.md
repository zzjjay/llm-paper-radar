#### 摘要
针对多 adapter LLM serving 中切换 adapter 时 KV cache 无法复用、需大量重算的问题，论文基于 Activated LoRA (aLoRA) 在 vLLM 中实现 base 与 adapted 模型间的 prefix cache 跨模型复用。核心是在模型执行路径中引入 base-aligned block hashing 与 activation-aware masking，使 cache 可跨模型共享且兼容现有 serving 优化。在多轮多 adapter pipeline 上相比标准 LoRA 取得最高 58x 端到端延迟下降、>100x TTFT 改善，收益随模型规模和序列长度扩大。
