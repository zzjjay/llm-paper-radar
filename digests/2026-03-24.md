#### 摘要
SANA-Video 是一个高效视频生成扩散模型，支持最高 720x1280 分辨率和分钟级时长的视频生成。核心设计有两点：用 linear attention 替代 vanilla attention 以处理视频生成中大规模 token，以及基于 linear attention 累加性质设计 constant-memory KV cache 的 block-wise autoregressive 方案，以固定内存代价提供全局上下文。训练成本仅 64 张 H100 GPU 训练 12 天（约为 MovieGen 的 1%），在 RTX 5090 上以 NVFP4 精度部署可将 5 秒 720p 视频生成延迟从 71s 降至 29s（2.4x 加速），同时推理速度比同量级模型 Wan 2.1-1.3B 和 SkyReel-V2-1.3B 快 16x。

- 🎯 用 linear attention 构建 Linear DiT，并设计 constant-memory KV cache 的 block-wise autoregressive 方案支持分钟级长视频生成
- 📊 训练成本仅 64×H100 GPU × 12 天，约为 MovieGen 的 1%
- 📊 RTX 5090 上 NVFP4 精度部署，5 秒 720p 视频生成从 71s 降至 29s（2.4x 加速），比同量级 SOTA 模型快 16x
- 💡 利用 linear attention 的累加性质实现固定内存代价的全局上下文，无需传统 KV cache
