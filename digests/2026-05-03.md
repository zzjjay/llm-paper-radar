#### 摘要
SANA-Video 针对高分辨率长视频生成的效率瓶颈，提出两个核心设计：以 linear attention 替代 vanilla attention 构建 Linear DiT，以及基于 linear attention 累积性质的 constant-memory KV cache，使显存开销固定且支持分钟级视频生成。训练成本仅为 MovieGen 的 1%（64 张 H100 跑 12 天），与同量级模型 Wan 2.1-1.3B 和 SkyReel-V2-1.3B 相比性能相当但推理速度快 16x。在 RTX 5090 上以 NVFP4 精度部署后，生成 5 秒 720p 视频从 71s 降至 29s，加速 2.4x。

- 🎯 Linear DiT + constant-memory KV cache，固定显存代价实现分钟级长视频生成
- 📊 推理速度比 Wan 2.1-1.3B / SkyReel-V2-1.3B 快 16x，训练成本仅为 MovieGen 的 1%
- 📊 RTX 5090 上 NVFP4 部署，720p 5秒视频生成从 71s→29s，加速 2.4x
- 🔧 支持 720×1280 分辨率、分钟级时长，可在消费级 GPU 上部署
