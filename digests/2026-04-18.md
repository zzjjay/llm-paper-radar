- 📊 在多种模型和任务上实现 >6x lossless 加速，比 EAGLE-3 快最多 2.5x

#### 🧭 为什么推送这篇
**Diffusion 压缩** · topic 4/5 · practicality 4/5 · composite 8/10 · 扩散模型压缩工作，采用线性注意力和常量内存 KV 缓存设计提高视频生成效率，部署 NVFP4 量化实现 2.4 倍推理加速。核心贡献是模型架构设计与量化部署的结合，而非单纯的量化方法论。评估依赖视觉质量对比而非标准基准，缺乏精确的定量精度指标。

#### 摘要
SANA-Video 针对视频生成中 token 数量庞大导致的计算和内存瓶颈，提出两项核心设计：以 linear attention 替代 vanilla attention 的 Linear DiT，以及利用 linear attention 累积性质构造的 constant-memory KV cache，支持分块自回归生成分钟级长视频且内存开销固定。训练成本仅为 MovieGen 的 1%（64 张 H100 GPU 训练 12 天），在 720x1280 分辨率下与 Wan 2.1-1.3B、SkyReel-V2-1.3B 等同规模 SOTA 模型性能相当，推理速度快 16x。在 RTX 5090 上以 NVFP4 精度部署，生成 5 秒 720p 视频从 71s 加速至 29s（2.4x speedup）。

- 🎯 Block Linear Attention 的 constant-memory KV cache 消除传统 KV cache，支持固定内存代价的分钟级视频生成
- 📊 训练成本仅为 MovieGen 的 1%，推理延迟比同规模 SOTA 模型快 16x
- 📊 RTX 5090 + NVFP4 部署将 720p 5 秒视频生成从 71s 降至 29s（2.4x speedup）
- 🔧 可在单张 RTX 5090 消费级 GPU 上部署，支持 720x1280 高分辨率生成

#### 📎 相关方法 / 对比基线
