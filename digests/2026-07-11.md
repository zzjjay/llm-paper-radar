#### 摘要
LLM 在线推理需要大批量并发请求，但 KV cache 内存因碎片化和冗余复制而严重浪费，限制了有效 batch size。PagedAttention 借鉴操作系统的虚拟内存与分页机制，将 KV cache 切分为固定大小的 block 并按需映射，实现近零内存浪费以及请求内/跨请求的灵活 KV cache 共享。基于此构建的 vLLM 系统在相同延迟水平下，吞吐量相比 FasterTransformer 和 Orca 提升 2–4×，在长序列、大模型和复杂解码算法场景下提升更为显著。

- 🎯 PagedAttention 将 KV cache 按 OS 分页机制管理，消除碎片与冗余复制，实现近零内存浪费
- 📊 vLLM 在相同延迟下吞吐量比 FasterTransformer 和 Orca 提升 2–4×
- 💡 支持请求内及跨请求 KV cache 灵活共享，可优化 parallel sampling、beam search 等场景
- 🔧 源码已开源，vLLM 已被大规模实际部署
