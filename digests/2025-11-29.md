- 🎯 方法：用 block diffusion 模型替代自回归 draft，单次 forward pass 并行生成 draft tokens
- 💡 创新：draft 模型复用目标模型提取的 context features，提升输出质量与 acceptance rate
- 📊 结果：多任务下实现 >6x lossless 加速，最高比 EAGLE-3 快 2.5x

#### 摘要
SANA-Video 是一个面向高效视频生成的小型扩散模型，支持最高 720x1280 分辨率和分钟级时长视频。核心设计包含两点：用 linear attention 替代 vanilla attention 以降低大量 token 的计算开销，以及基于 linear attention 累积特性设计的 constant-memory KV cache Block Linear Attention，以固定显存开销实现全局上下文建模，支持长视频自回归生成。模型在 64 张 H100 GPU 上训练仅需 12 天（约为 MovieGen 的 1%），与 Wan 2.1-1.3B 和 SkyReel-V2-1.3B 等同规模 SOTA 模型性能相当，推理延迟快 16x，并可在 RTX 5090 上以 NVFP4 精度部署，生成 5 秒 720p 视频从 71s 加速至 29s（2.4x）。

- 🎯 方法：Block Linear Attention 配合 constant-memory KV cache，固定显存实现分钟级长视频自回归生成
- 📊 结果：与 Wan 2.1-1.3B、SkyReel-V2-1.3B 性能相当，推理延迟快 16x
- 📊 结果：NVFP4 部署在 RTX 5090 上，5 秒 720p 视频生成从 71s 降至 29s（2.4x 加速）
- 🔧 工程：训练成本仅 64×H100 GPU × 12 天，约为 MovieGen 的 1%
