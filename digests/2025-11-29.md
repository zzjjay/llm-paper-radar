#### 摘要
针对推理 LLM 长序列下 KV cache 压缩中现有 token eviction 方法只依赖 local attention score、忽略 token 长期重要性的问题，提出 G-KV：在 decoding 阶段使用结合 local 与 historical attention score 的 global scoring 机制来评估 token 重要性并执行 eviction。同时引入 reinforcement learning 与 distillation 的 post-training 流程，使模型适配压缩后的 KV cache 设置。
