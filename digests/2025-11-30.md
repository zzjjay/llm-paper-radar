#### 摘要
SANA-Video 针对高分辨率（720x1280）、分钟级长视频生成的效率问题，提出两项核心设计：Linear DiT 以线性 attention 替代 vanilla attention 降低大量 token 处理的计算开销，以及基于线性 attention 累积性质的 Constant-Memory KV cache，以固定显存代价为 Block Linear Attention 提供全局上下文，消除传统 KV cache 随序列增长的显存需求。训练仅需 64 张 H100 GPU 跑 12 天（约为 MovieGen 的 1%），在 RTX 5090 上以 NVFP4 精度将 5 秒 720p 视频生成延迟从 71s 降至 29s（2.4x 加速），同时相比 Wan 2.1-1.3B 和 SkyReel-V2-1.3B 等小模型实现 16x 延迟加速。

- 🎯 用线性 attention 构建 Linear DiT，配合 Constant-Memory KV cache 实现固定显存长视频自回归生成

- 📊 与同量级模型（Wan 2.1-1.3B、SkyReel-V2-1.3B）相比延迟快 16x
