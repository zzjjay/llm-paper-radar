#### 摘要
KV cache 的内存带宽和占用是长上下文自回归推理的瓶颈。OCTOPUS 在 TurboQuant/PolarQuant 等 rotation-preconditioned codec 的基础上，对旋转后的坐标三元组进行联合量化：利用八面体参数化将每个三元组的方向映射到正方形，再对两个坐标和三元组范数分别进行 Lloyd-Max 量化，从而得到严格非均匀的 per-triplet bit allocation。该 codec 无需校准数据，在线且确定性可复现，在文本、视频、音频任务上以低 bit width 超越所有已报告的 rotation codec，并通过 fused Triton kernel 实现 on-the-fly key 重建，不引入额外带宽或延迟。

- 🎯 方法：对旋转 KV 坐标三元组联合量化，用八面体参数化+Lloyd-Max 得到非均匀 bit allocation
- 💡 创新：data-oblivious、online、deterministic，无需 calibration 数据，仅依赖 key 总维度
- 📊 结果：在所有报告 bit width 和指标上匹配或超越所有 rotation codec，极低 bit 下优势更显著
- 🔧 工程：fused Triton 实现 on-the-fly key 重建，零额外解码带宽与延迟
