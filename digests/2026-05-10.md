#### 摘要
SANA-Video 针对高分辨率长视频生成的效率瓶颈，提出两项核心设计：用 linear attention 替代 vanilla attention 以降低大 token 数下的计算复杂度，并设计基于 linear attention 累积特性的 constant-memory KV cache，以固定显存代价实现块状自回归长视频生成。训练成本仅需 64 张 H100 GPU 训练 12 天（MovieGen 的 1%），在 720x1280 分辨率下生成速度比 Wan 2.1-1.3B 和 SkyReel-V2-1.3B 快 16x，部署于 RTX 5090 并使用 NVFP4 精度后，生成 5 秒 720p 视频从 71s 降至 29s（2.4x 加速）。

- 🎯 Block Linear Attention 利用 linear attention 的累积特性，以 constant-memory KV cache 替代传统 KV cache 实现分钟级长视频生成
- 📊 与 Wan 2.1-1.3B、SkyReel-V2-1.3B 性能相当，推理延迟快 16x
- 📊 RTX 5090 上使用 NVFP4 精度，720p 5 秒视频生成从 71s 加速至 29s（2.4x）
- 🔧 训练总成本 64×H100 GPU × 12 天，仅为 MovieGen 的 1%
