#### Summary

#### Summary

Quant.npu is an integer-only fully static quantization framework using learnable quantization parameters and rotation matrices to avoid runtime parameter re-computation. On real mobile NPUs it matches SOTA accuracy while cutting inference latency by up to 15.1%.

- 🎯 Integer-only fully static quantization eliminates runtime quantization parameter recomputation

#### Summary
For long-context and multi-modal LLMs, KV cache memory dominates and per-channel quantization collapses under extreme bit-widths. The paper identifies Token Norm Imbalance (TNI) as the root cause: shared quantization parameters cannot span tokens with disparate norms. OScaR augments the per-channel paradigm with Canalized Rotation followed by Omni-Token Scaling to suppress sequence-dimensional variance, backed by optimized CUDA kernels. Across text-only, multi-modal, and omni-modal LLMs, OScaR is near-lossless at INT2, delivers up to 3.0x decoding speedup, 5.3x memory reduction, and 4.1x throughput over the BF16 FlashDecoding-v2 baseline.

- 📊 Up to 3.0x decoding speedup, 5.3x memory reduction, 4.1x throughput vs BF16 FlashDecoding-v2

#### 📎 Related methods / baselines
