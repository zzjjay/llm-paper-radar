#### 摘要
SANA-Video 针对高分辨率长视频生成的效率问题，提出两项核心设计：Linear DiT（以 linear attention 替代 vanilla attention 降低 token 处理开销）和 Constant-Memory KV cache for Block Linear Attention（利用 linear attention 的累积性质实现固定显存的 block-wise autoregressive 长视频生成）。训练成本压缩至 64 张 H100 GPU 上 12 天，仅为 MovieGen 的 1%。与 Wan 2.1-1.3B、SkyReel-V2-1.3B 等同量级模型相比，推理延迟降低 16x；在 RTX 5090 以 NVFP4 精度部署时，生成 5 秒 720p 视频从 71s 加速至 29s（2.4x speedup）。

- 🎯 Block Linear Attention 的 Constant-Memory KV cache，固定显存实现 minute-level 长视频自回归生成
- 📊 训练成本仅为 MovieGen 的 1%（64 H100 GPU × 12 天），推理速度比同级模型快 16x
- 📊 RTX 5090 + NVFP4 推理，5 秒 720p 视频从 71s 降至 29s，加速 2.4x
- 🔧 可在单张 RTX 5090 GPU 上部署，支持 720×1280 分辨率及分钟级时长视频
