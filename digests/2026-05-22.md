#### 摘要
针对 LLM serving 中 KV cache 显存碎片化和冗余复制限制 batch size 的问题，提出受操作系统虚拟内存与分页启发的 PagedAttention 算法，将 KV cache 按 block 管理。基于此构建 vLLM 服务系统，实现 KV cache 近零浪费以及请求内/跨请求的灵活共享。在同等 latency 下，相比 FasterTransformer 与 Orca 等 SOTA 系统，主流 LLM 吞吐提升 2-4×，长序列、大模型、复杂解码算法下增益更显著。
