#### 摘要
SANA-Video 是一个面向高效视频生成的小型 diffusion model，支持最高 720×1280 分辨率和分钟级时长的视频合成。核心设计有两点：（1）用 linear attention 替代 vanilla attention 作为 Linear DiT 的基础操作，降低大 token 数下的计算开销；（2）为 block-wise autoregressive 长视频生成设计了 constant-memory KV cache，利用 linear attention 的累加性质以固定显存提供全局上下文，无需传统 KV cache。训练成本压缩至 64 张 H100 上 12 天（仅为 MovieGen 的 1%），在同量级小模型中实现 16× 推理加速，并可在 RTX 5090 上以 NVFP4 精度将 5 秒 720p 视频的生成延迟从 71s 降至 29s。

- 🎯 Linear DiT + constant-memory block linear attention KV cache，支持分钟级长视频生成
- 📊 推理延迟比 Wan 2.1-1.3B / SkyReel-V2-1.3B 快 16×，质量保持竞争力
- 📊 NVFP4 精度在 RTX 5090 上推理 5s 720p 视频从 71s 降至 29s（2.4× speedup）
- 🔧 训练成本仅 64×H100 × 12 天，约为 MovieGen 的 1%
