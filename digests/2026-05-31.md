#### 摘要
针对长上下文 LLM serving 中 INT2 KV cache 量化精度崩溃的问题，OSCAR 离线估计 attention-aware 的协方差结构，据此导出固定 rotation 和 clipping 阈值，使量化方向与下游 attention 实际消费的协方差对齐。配套实现了与 paged KV cache 和 fused kernel 兼容的自定义 INT2 attention kernel，可直接接入 SGLang/vLLM。在 Qwen3-4B/8B/32B 与 GLM-4.7 (358B) 上，相对 BF16 精度差分别压到 3.78 / 1.42 点等水平，而 naive rotation INT2 几乎崩溃；128K RULER-NIAH 长上下文同样保持稳健。系统侧 KV cache 显存减少约 8x，大 batch 吞吐最高提升 7x，batch=1 解码最高加速 3x。
