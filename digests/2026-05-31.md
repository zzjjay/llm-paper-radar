#### 摘要
针对长上下文 LLM serving 中 INT2 KV cache 量化精度崩溃的问题，OSCAR 离线估计 attention-aware 的协方差结构，据此导出固定 rotation 与 clipping 阈值，使量化方向与下游 attention 实际消费的协方差对齐。配套实现兼容 paged KV cache 与 fused kernel 的自定义 INT2 attention kernel，可直接接入 SGLang 与 vLLM。在 Qwen3-4B-Thinking-2507 与 Qwen3-8B 上将与 BF16 的精度差压到 3.78 与 1.42 点（朴素 rotation INT2 几乎归零），并在 Qwen3-32B 和 GLM-4.7 (358B) 上保持与 BF16 相当；RULER-NIAH 128K 长上下文下依然稳健。系统侧 KV cache 内存约降 8x，大 batch 吞吐最高提升 7x，batch=1 解码最高加速 3x。
