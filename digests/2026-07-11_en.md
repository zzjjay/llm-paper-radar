#### Summary
High-throughput LLM serving requires large request batches, but existing systems waste significant KV cache memory through fragmentation and redundant duplication, constraining batch size. PagedAttention adapts classical OS virtual memory and paging techniques to partition KV cache into fixed-size blocks mapped on demand, achieving near-zero waste and enabling flexible KV cache sharing within and across requests. Built on PagedAttention, vLLM delivers 2–4× throughput improvement over FasterTransformer and Orca at equivalent latency levels, with larger gains on longer sequences, bigger models, and more complex decoding algorithms.

- 🎯 PagedAttention manages KV cache via OS-inspired paging, eliminating fragmentation and redundant duplication
