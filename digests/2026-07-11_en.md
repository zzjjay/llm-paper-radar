#### Summary
High-throughput LLM serving requires large batches, but KV cache memory is bulky and dynamically sized, causing severe waste from fragmentation and redundant duplication in existing systems. PagedAttention adapts OS virtual-memory paging to partition KV cache into fixed-size blocks, enabling near-zero waste and flexible cross-request sharing. Built on PagedAttention, vLLM achieves 2–4× throughput improvement over state-of-the-art systems such as FasterTransformer and Orca at the same latency level, with gains amplifying for longer sequences, larger models, and more complex decoding algorithms.

- 🎯 PagedAttention partitions KV cache into fixed blocks using OS-inspired paging, achieving near-zero memory fragmentation
- 📊 2–4× throughput improvement over FasterTransformer and Orca at equivalent latency
- 💡 Enables flexible intra- and inter-request KV cache sharing, reducing memory for beam search and parallel sampling
