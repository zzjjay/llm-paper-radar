#### 摘要
针对长上下文 LLM 推理中 KV cache 占用 GPU 显存、offload 到 DRAM 后受限于 GPU-CPU 传输或 CPU 算力的问题，ScoutAttention 设计了 GPU-CPU 协同的 block-wise sparse attention 来降低 CPU 负载。核心创新是 layer-ahead CPU pre-computation 算法，让 CPU 提前一层启动 attention 计算，并配合异步周期性 recall 机制保持低 CPU 压力。实验显示精度损失控制在 2.4% 以内，相比现有 offloading 方法达到 2.1x 加速。
