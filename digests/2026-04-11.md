#### 摘要
SANA-Video 针对高分辨率长视频生成中 attention 计算和 KV cache 内存开销过大的问题，提出 Linear DiT 架构，以 linear attention 替换 vanilla attention 降低大 token 数下的计算复杂度。核心贡献是基于 linear attention 累积性质设计的 constant-memory KV cache Block Linear Attention，以固定内存开销实现分钟级长视频的块自回归生成。模型训练仅需 64 张 H100 GPU 运行 12 天（MovieGen 训练成本的 1%），生成 720x1280 分辨率视频时比 Wan 2.1-1.3B 和 SkyReel-V2-1.3B 快 16×，在 RTX 5090 上以 NVFP4 精度推理 5 秒 720p 视频从 71s 加速至 29s（2.4× 加速）。

- 🎯 linear attention 替代 vanilla attention 构建 Linear DiT，constant-memory KV cache 支持分钟级长视频块自回归生成
- 📊 训练成本仅为 MovieGen 的 1%（64 H100 GPU × 12 天），推理速度比同规模模型快 16×
- 📊 RTX 5090 上 NVFP4 精度推理 5 秒 720p 视频，延迟从 71s 降至 29s（2.4× 加速）
- 🔧 可部署于单张 RTX 5090 GPU，支持最高 720x1280 分辨率输出
