#### 🧭 为什么推送这篇
**KV cache 压缩** · topic 4/5 · practicality 4/5 · composite 8/10 · CodeComp是一种training-free的KV cache压缩框架，将静态程序分析（Code Property Graph via Joern）引入LLM推理，以识别代码中结构关键token（call sites、branch conditions、assignments等）并在KV cache eviction中优先保留它们。主要贡献是KV cache压缩算法，直接面向agentic coding场景下的内存瓶颈。evaluation涵盖bug localization和code generation基准，显示在aggressive压缩率下仍能接近full-context精度，并无缝集成到SGLang pipeline。方法简洁（training-free，仅需Joern静态分析作为prior），部署路径清晰。但largest model未明确说明，且无明确speedup/throughput数字，适当降分。

#### 摘要
针对 agentic code 任务中长代码库导致的 KV cache 显存瓶颈，现有压缩方法仅依赖 attention signals 估计 token 重要性，会丢弃 call sites、branch conditions、assignments 等结构关键 token。CodeComp 是 training-free 的 KV cache 压缩框架，将 Joern 提取的 Code Property Graph 静态分析先验注入 LLM 推理。在 bug localization 和 code generation benchmark 上等显存预算下持续超越 attention-only baseline，激进压缩下可恢复大部分 full-context 精度，patch 生成质量与 full-context 推理持平，并无缝集成到 SGLang agentic coding pipeline。
