#### Summary
The system targets low-latency serving of Qwen3.5-4B on a resource-constrained NVIDIA A10G GPU by combining a quantized target model with speculative decoding. Accuracy is recovered via quantization-aware distillation applied to the target model while retaining the original quantization grid. A block-diffusion drafter specialized for the quantized target is trained in two stages — first from the high-precision target, then adapted to the low-precision target — and is further optimized with quantization and sliding-window attention to reduce long-context decoding latency. The resulting submission achieves a 6.978× average speedup over the baseline while satisfying required quality thresholds, ranking 3rd overall in the competition.

#### Summary
Existing NVFP4 pretraining methods focus on transformer linear layers, leaving optimizer states and attention unstable in full 4-bit pipelines. Full-Stack FP4 is the first complete NVFP4 pretraining framework addressing all three stability bottlenecks: LoRA-SVD decomposition on linear projections reduces the linear-only loss gap from 1.40% to 0.61%; a designed second-moment transformation enables robust NVFP4 AdamW storage and supports native NVFP4 Newton-Schulz iterations for the Muon optimizer; a mixed-precision attention scheme quantizes Q/K/V and backward dS while guarding vulnerable paths in BF16. End-to-end pretraining of a 3B model on 64B tokens achieves a total loss gap of only 1.47% relative to BF16.

- 🎯 LoRA-SVD decomposition for linear projections shrinks the linear-only loss gap from 1.40% to 0.61%
- 🎯 NVFP4 second-moment transformation for AdamW and native NVFP4 Newton-Schulz for Muon optimizer resolve optimizer numerical instability
- 📊 3B model / 64B token end-to-end NVFP4 pretraining achieves only 1.47% loss gap vs. BF16 baseline
- 💡 First framework covering linear projections, optimizer states, and attention simultaneously in NVFP4; all modules are plug-and-play

- AdamW — second moments redesigned with NVFP4 transformation for stable low-precision storage
