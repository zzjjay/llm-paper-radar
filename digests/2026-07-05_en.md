#### Summary
Existing NVFP4 pretraining methods focus on transformer linear layers, leaving optimizer states, optimizer arithmetic, and attention underexplored in 4-bit pipelines. Full-Stack FP4 addresses all three stability bottlenecks with module-wise precision strategies: LoRA-SVD decomposition for linear projections reduces the loss gap from 1.40% to 0.61%; AdamW second-moment transformation enables robust NVFP4 storage with native NVFP4 Newton-Schulz support for the Muon optimizer; a mixed-precision attention scheme quantizes Q/K/V and backward dS while guarding vulnerable paths (PV/dOV^T) in BF16 with unified tensor reuse for forward-backward alignment. Validated on 3B-parameter/64B-token pretraining, the end-to-end loss gap reaches only 1.47% relative to BF16, demonstrating the first stable full-stack NVFP4 LLM pretraining framework.

- 📊 3B model trained on 64B tokens achieves only 1.47% end-to-end loss gap vs. BF16 baseline
