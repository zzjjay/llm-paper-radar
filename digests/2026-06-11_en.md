#### Summary
Applying NVFP4 to large reasoning models (LRMs) faces two issues: reasoning accuracy degrades under quantization, and existing NVFP4 kernels fail to realize latency gains in small-batch autoregressive decoding. Analyzing token-level uncertainty, the authors find quantization increases incorrect sampling at low-entropy symbolic tokens while over-concentrating on few tokens in high-uncertainty steps. #### Summary
Drop-by-Drop addresses inference-time precision control over LLM weights across heterogeneous hardware without retraining. The method is grounded in information-theoretic successive refinement, showing that Gaussian-distributed LLM weights can be optimally reconstructed with increasing fidelity as bits are added under a weighted MSE distortion. It injects Matryoshka-style supervision into the loss and exploits additive codebook structure so that ordered codebook subsets yield accurate partial reconstructions at each precision level. A single checkpoint thus serves multiple bitwidths, cutting storage and memory overhead while maintaining competitive perplexity and accuracy on Qwen, LLaMA, Gemma, and Mistral.

#### Summary
TWLA is a PTQ framework targeting W1.58A4 for LLMs (1.58-bit ternary weights plus 4-bit activations), addressing the fact that existing ternarization methods keep activations in high precision due to heavy-tailed distributions, which limits end-to-end inference acceleration. It has three parts: E2M-ATQ minimizes layer-output error via two-stage optimization from Euclidean initialization to manifold relocation; KOTMS applies a Kronecker-structured orthogonal rotation to reshape weights into ternary-friendly tri-modal distributions while statistically suppressing activation outliers; ILA-AMP explicitly adds adjacent-layer second-order interaction costs into bit allocation to prevent cascades from weak layers. Experiments show TWLA keeps high accuracy under W1.58A4 while delivering significant inference acceleration; code is released.
