#### 摘要
SANA-Video 针对高分辨率、长时视频生成的效率瓶颈，提出基于 Linear DiT 的扩散模型。核心创新有两点：用 linear attention 替代 vanilla attention 以处理视频 token 数量激增的问题，以及基于 linear attention 累积特性设计的 constant-memory KV cache，使长视频 block-wise autoregressive 生成的显存开销固定不增长。模型在 64 H100 GPU 上仅训练 12 天（约为 MovieGen 成本的 1%），生成性能与 Wan 2.1-1.3B、SkyReel-V2-1.3B 相当，但实测延迟快 16x；在 RTX 5090 上以 NVFP4 精度推理 5 秒 720p 视频从 71s 降至 29s（2.4x 加速）。

- 🎯 constant-memory KV cache 利用 linear attention 累积性质，固定显存实现 minute-level 长视频生成
- 📊 对比 Wan 2.1-1.3B / SkyReel-V2-1.3B，性能相当但实测延迟快 16x
- 📊 RTX 5090 NVFP4 推理 5s 720p 视频：71s → 29s，2.4x 加速
- 🔧 训练成本 64 H100 GPU × 12 天，仅为 MovieGen 的 1%
