**KV cache 压缩** · topic 4/5 · practicality 3/5 · composite 7/10 · 论文提供了对注意力模式的统一理论框架（TAPPA），指导KV缓存压缩和剪枝。但缺乏实验评估细节：未报告具体模型大小、准确性基准、推理加速数字。虽然应用于两个压缩任务，但论文重点在分析理论而非实用部署。主要贡献是理解而非新颖压缩算法。KV缓存压缩是核心且符合scope，但评估不充分导致practicality偏低。

#### 摘要
现有对 LLM attention patterns（retrieval heads、sink heads、diagonal traces 等）的观察彼此孤立，缺乏统一解释。TAPPA 框架从时间连续视角对 attention patterns 的数学形式进行分析，将其划分为「可预测 patterns」和「不可预测 patterns」，并证明这一区分可由 query 在时间维度上的自相似性程度解释。通过对 queries、keys 和 RoPE 的联合数学分析，TAPPA 推导出三类典型可预测 pattern 的成因，并将其洞察应用于 KV cache compression 和 LLM pruning，所提 metric 在两类任务上均超越 baseline 方法。

- 💡 提出 TAPPA 框架，用 query 时间维度自相似性统一解释多种 attention patterns 的成因
- 🎯 将 attention patterns 分为可预测（有明确规律）和不可预测（近似随机）两类，并给出数学推导
- 🔧 基于 TAPPA 设计简单 metric，在 KV cache compression 与 LLM pruning 任务上持续超越 baseline
