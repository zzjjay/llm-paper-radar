**Survey / 方法论与对比** · topic 3/5 · practicality 2/5 · composite 5/10 · TAPPA是一个分析框架，通过时间连续视角统一解释attention patterns（retrieval heads、sink heads、diagonal traces等），并将insight应用于KV cache压缩和LLM pruning。该工作主要贡献是机制分析和理论统一，而非提出新的压缩算法，属于survey/methodology范畴。但其primary contribution更偏向机制分析（attention pattern可预测性）而非头对头压缩方法的对比测量，survey价值有限。应用部分仅展示一个简单metric'consistently improves performance over baseline'，缺乏具体benchmark名称、模型规模和定量数据，评估深度不足。综合来看，topic_relevance为3（间接相关，分析框架+简单应用），practicality为2（metric本身简单，但缺乏具体部署数据和详细评估）。

#### 摘要
针对 LLM 中 retrieval heads、sink heads、diagonal traces 等 attention pattern 缺乏统一解释的问题，提出 Temporal Attention Pattern Predictability Analysis (TAPPA) 框架，从时间连续视角分析 attention 的数学形式。TAPPA 将 attention pattern 分为可预测（有清晰规律）和不可预测（近似随机）两类，并指出区分依据是 query 在时间维度上的 self-similarity 程度；进一步通过 query、key 和 RoPE 的联合作用对三类可预测模式做数学推导。基于 TAPPA 设计的简单度量在 KV cache compression 和 LLM pruning 任务上均优于 baseline。

- 💡 用 query 时间 self-similarity 解释 attention pattern 可预测性
- 🎯 统一分析 retrieval/sink/diagonal heads，含 RoPE 联合推导
- 📊 TAPPA 度量在 KV cache 压缩与剪枝上均超 baseline
