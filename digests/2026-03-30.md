#### 摘要
SANA-Video 针对高分辨率长视频生成的效率瓶颈，提出两项核心设计：用 linear attention 替代 vanilla attention 以处理视频 token 数量庞大的问题，以及基于 linear attention 累积性质设计的 constant-memory KV cache block-wise autoregressive 方案，使显存开销固定化并支持分钟级长视频生成。训练仅需 64 张 H100 GPU 运行 12 天（约为 MovieGen 的 1%）。在 RTX 5090 上以 NVFP4 精度部署，生成 5 秒 720p 视频从 71s 降至 29s（2.4x 加速），且与 Wan 2.1-1.3B、SkyReel-V2-1.3B 等同规模模型相比延迟低 16x。

- 🎯 Constant-memory KV cache 利用 linear attention 累积特性，以固定显存代价提供全局上下文，支持分钟级视频生成
- 📊 RTX 5090 上 NVFP4 推理 5 秒 720p 视频：71s→29s，2.4x 加速
- 📊 训练成本仅 64×H100×12 天，约为 MovieGen 的 1%，生成延迟比 Wan 2.1-1.3B 和 SkyReel-V2-1.3B 快 16x
- 💡 Linear DiT 以 linear attention 为核心算子，相比 vanilla attention 在大 token 量视频场景下更高效
