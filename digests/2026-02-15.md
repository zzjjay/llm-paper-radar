#### 🧭 为什么推送这篇
**Diffusion 压缩** · topic 4/5 · practicality 4/5 · composite 8/10 · 扩散模型压缩核心贡献：线性注意力替代标准注意力以处理大量视频令牌；恒定内存块线性注意力KV缓存设计实现长视频高效生成；NVFP4量化部署提供2.4x推理加速。虽然未报告标准视频压缩度量（FID、帧PSM等），但VBench评估和与SOTA小型扩散模型的竞争性对标、加上显著的推理加速和内存效率改进，使其符合扩散压缩主题。

#### 摘要
SANA-Video 针对高分辨率长视频生成中 attention 计算和 KV cache 内存开销过大的问题，提出两项核心设计：Linear DiT（以线性 attention 替代 vanilla attention 处理大量 token）和 Block Linear Attention 的常量内存 KV cache（利用线性 attention 的累积性质实现固定内存代价的全局上下文）。训练成本仅需 64 张 H100 GPU 跑 12 天（约为 MovieGen 的 1%），可生成 720x1280 分辨率、分钟级时长视频。与同规模模型 Wan 2.1-1.3B 和 SkyReel-V2-1.3B 相比性能相当，但推理速度快 16x；在 RTX 5090 以 NVFP4 精度部署后，生成 5 秒 720p 视频从 71s 降至 29s（2.4x 加速）。

- 🎯 Block Linear Attention 使用常量内存 KV cache，固定内存代价实现分钟级长视频自回归生成
- 📊 推理延迟比 Wan 2.1-1.3B、SkyReel-V2-1.3B 快 16x，训练成本仅为 MovieGen 的 1%
- 📊 RTX 5090 + NVFP4 部署后，5 秒 720p 视频生成从 71s 加速至 29s（2.4x）
- 🔧 可在单张 RTX 5090 GPU 上部署，支持 720x1280 分辨率及分钟级视频
