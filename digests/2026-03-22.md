#### 摘要
SANA-Video 是一个小型扩散模型，目标是在消费级硬件上高效生成最高 720x1280 分辨率、分钟级时长的视频。核心方法有两个：（1）以 linear attention 替换 vanilla attention 构建 Linear DiT，降低大量 token 处理的计算开销；（2）基于 linear attention 的累积性质设计 Constant-Memory KV cache，支持 block-wise 自回归长视频生成，内存占用固定不随长度增长。训练成本仅为 MovieGen 的 1%（64 张 H100 GPU 训练 12 天），在与 Wan 2.1-1.3B 和 SkyReel-V2-1.3B 竞争的质量下推理速度快 16x，并可在 RTX 5090 上以 NVFP4 精度将 5 秒 720p 视频的生成延迟从 71s 压缩到 29s。

- 🎯 Linear DiT + Constant-Memory KV cache 实现固定内存的 block-wise 自回归长视频生成
- 📊 推理速度比 Wan 2.1-1.3B 和 SkyReel-V2-1.3B 快 16x，质量持平
- 📊 RTX 5090 上用 NVFP4 精度推理，5 秒 720p 视频从 71s 加速至 29s（2.4x speedup）
- 🔧 训练成本仅需 64 H100 GPU × 12 天，为 MovieGen 训练成本的 1%
