#### 摘要
针对现有 layer-wise pruning 依赖静态启发式规则、忽略层间依赖的问题，本文将层剪枝建模为合作博弈，每层作为玩家、模型性能作为效用函数，用 Shapley value 衡量各层边际贡献。由于 LLM 上精确计算 Shapley value 不可行，作者训练一个轻量 surrogate network 来预测任意层组合下的 LLM 性能，并配合 stratified Monte Carlo mask sampling 降低估计成本。实验显示在 perplexity 和 zero-shot accuracy 上均优于现有 layer pruning 方法。

- 💡 把 layer pruning 建模为 cooperative game，用 Shapley value 量化层贡献
- 🎯 surrogate network 预测任意层组合性能，避开真实 LLM 评估开销
- 🔧 stratified Monte Carlo mask sampling 进一步降低 Shapley 估计成本
- 📊 PPL 与 zero-shot accuracy 一致超越静态启发式 pruning
