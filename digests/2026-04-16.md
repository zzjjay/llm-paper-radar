#### 🧭 为什么推送这篇
**Survey / 方法论与对比** · topic 4/5 · practicality 3/5 · composite 7/10 · 本文系统研究了FP16 KV cache推理与无cache重计算之间的数值等价性问题，发现由于FP16非结合性，两者在所有模型和采样策略下均出现100%的token divergence。研究属于measurement/methodology类工作，无新算法提出，适合归入survey桶。结论对KV cache量化工程师有直接价值：在部署FP8/INT8/MXFP4 KV cache方案时，理解FP16基线本身并非数值稳定参考这一事实，影响到校准、基线对比和精度回归分析。但最大测试模型仅7B，且属于纯数值分析，无法指导具体压缩算法的选择，practicality有限。

#### 摘要
本文揭示 FP16 精度下 KV cache 推理与无 cache 重计算并非数值等价：两者采用不同的浮点累加顺序，因 FP16 非结合性产生确定性 token 序列分歧。在 LLaMA-2-7B、Mistral-7B-v0.3、Gemma-2-2B 三个模型的 GSM8K 评测中，包括 greedy decoding 在内的所有采样策略均出现 100% token 分歧率，且 cache-ON 在 9 个条件中 8 个精度更高，表明分歧方向是系统性的。FP32 对照将分歧降低 8 个数量级，flip rate 降至 0.0%，确认 FP16 非结合性是唯一成因。逐层 drift profiling 显示 GQA 模型在首层即出现剧烈分歧，而 Gemma 的较大 head dimension 与 sliding window attention 则跨层均匀累积；对整个 residual stream 的 activation patching 无法恢复 cache-free 轨迹，将因果变量定位于 stateful KV cache 本身。

- 📊 GSM8K 上 greedy decoding 也出现 100% token 分歧率

- 💡 GQA 模型首层即剧烈分歧，Gemma 因 sliding window 跨层均匀
- 🎯 activation patching 整个 residual stream 无法恢复 cache-free 轨迹
