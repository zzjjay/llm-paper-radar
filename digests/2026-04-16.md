#### 摘要
本文揭示了 FP16 精度下 KV cache 推理与无 cache 重算在数值上并不等价：两者使用不同的浮点累加顺序，由于 FP16 非结合性，会在解码 token 序列上产生确定性偏离。在 LLaMA-2-7B、Mistral-7B-v0.3、Gemma-2-2B 三个模型上对 GSM8K 评测显示 100% token 分歧率，且 9 种条件中 8 种 cache-ON 反而精度更高。FP32 对照实验将分歧降低 8 个数量级并把 flip rate 降为 0.0%，证明 FP16 非结合性是唯一成因；逐层 drift 分析显示 GQA 模型在第一层即出现锐利分歧，而 Gemma 的大 head dim 与 sliding window attention 则使累加跨层均匀分布。对整个 residual stream 的 activation patching 无法恢复无 cache 轨迹，将因果变量定位到 stateful KV cache 本身。

- 🎯 residual stream patching 失败，因果定位于 stateful KV cache
