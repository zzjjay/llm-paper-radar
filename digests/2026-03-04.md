#### 🧭 为什么推送这篇
**Diffusion 压缩** · topic 4/5 · practicality 4/5 · composite 8/10 · 扩散模型压缩工作，结合线性注意力、块状自回归KV缓存设计和NVFP4量化。虽未报告精确的质量指标（FVD/LPIPS数值），但展现显著推理加速（2.4x）和部署友好性。算法相对简洁（线性注意力替代vanilla）且实现成本低（12天训练vs MovieGen的1200天）。评分稍低于满分是因为定性评价为主，缺乏量化精度

#### 摘要
SANA-Video 针对高分辨率长视频生成的效率瓶颈，提出 Block Linear Diffusion Transformer 架构。核心设计有两点：一是用 linear attention 替代 vanilla attention 以降低大 token 数下的计算复杂度；二是基于 linear attention 的累积性质设计 constant-memory KV cache，以固定显存代价为自回归长视频生成提供全局上下文。训练成本仅为 MovieGen 的 1%（64 H100 GPU 上 12 天），推理速度比同量级模型 Wan 2.1-1.3B 和 SkyReel-V2-1.3B 快 16x；在 RTX 5090 上以 NVFP4 精度部署后，生成 5 秒 720p 视频从 71s 加速至 29s（2.4x speedup）。

- 🎯 用 linear attention 构建 Linear DiT，并设计 constant-memory KV cache 支持 minute-level 长视频生成
- 📊 推理延迟比 Wan 2.1-1.3B、SkyReel-V2-1.3B 快 16x，同时保持竞争性生成质量
- 📊 RTX 5090 上 NVFP4 部署，720p 5s 视频推理从 71s 降至 29s（2.4x speedup）
- 📊 训练成本仅为 MovieGen 的 1%（64 H100 GPU × 12 天）
