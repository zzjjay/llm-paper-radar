#### 摘要
On-policy distillation (OPD) 已成为 LLM post-training 的核心技术，但其训练动态机制尚不清楚。本文系统分析 OPD 成功与失败的条件：student 与 teacher 需共享兼容的 thinking patterns，且 teacher 必须提供超出 student 已见能力的真正新知识；通过 weak-to-strong reverse distillation 验证，同家族 1.5B 和 7B teacher 在 student 视角下分布上无法区分。Token 级机制分析表明，成功的 OPD 以 high-probability tokens 上的渐进对齐为特征，一个小型共享 token 集合集中了 97%-99% 的概率质量；并进一步提出 off-policy cold start 和 teacher-aligned prompt selection 两种修复失败 OPD 的实用策略。

- 🎯 方法：OPD 成功需满足两条件：student/teacher thinking patterns 兼容，且 teacher 提供超出 student 已有能力的新知识
- 💡 创新：Token 级分析揭示成功 OPD 中共享 token 集集中 97%-99% 概率质量，呈现 progressive alignment 特征
- 🔧 工程：提出 off-policy cold start 与 teacher-aligned prompt selection 两种策略修复失败 OPD
- ⚠️ 局限：OPD 的 dense token-level reward 并非免费午餐，能否扩展至 long-horizon distillation 存疑
