#### 摘要
本文在 LoRA-adapted transformer 中研究 sub-token 粒度的路由，作为比 token / page / head / layer 更细的压缩轴。提出两种设定：query-independent 下结合 routed subspace LoRA 与 KV path 上的 value-group routing 做压缩感知语言建模；query-aware 下用 predictor-based selector 在 context-token/value-group pair 上按 query 相关性分配全局保留预算。实验表明前者在低 KV 预算下提升语言建模质量，后者在 KV 压缩下保持下游任务表现，且 sub-token routing 与 token-level query-aware selection 互补结合可在几乎不损精度下实现更深的 KV 压缩。

- 🎯 query-independent: routed LoRA + value-group routing
- 🎯 query-aware: predictor 按 query 分配 KV 保留预算
- 📊 与 token-level 选择互补，深度 KV 压缩近乎无损
