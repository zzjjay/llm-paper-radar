#### 摘要
本文在 LoRA-adapted transformer 中研究 sub-token 级别的路由压缩，作为比 token/page/head/layer 更细粒度的压缩轴。提出两种设计：query-independent 设置下将 routed subspace LoRA 与 KV path 上的 value-group routing 结合，用于压缩感知的语言建模；query-aware 设置下用 predictor-based selector 基于 query 相关性在 context-token/value-group 对上分配全局保留预算。结果显示前者在缩减 KV 预算下提升 LM 质量，后者在 KV 压缩下保持下游任务表现；二者与 token-level 选择互补，可在几乎不损失精度的情况下实现更深度的 KV 压缩。
