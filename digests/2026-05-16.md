#### 摘要
Layer pruning 移除整个 Transformer decoder block 后，存活层接收到的 hidden state 与其训练分布不匹配，导致性能大幅下降。Ghosted Layers 提出一种 training-free 的恢复模块，通过小规模 calibration set 推导出闭合解最优线性算子，重建被剪除层引入的 activation 偏差。该算子对应 alignment 目标的无约束最优解，而现有方法仅在受限算子子空间内求解约束最优。在多个 LLM backbone 和剪枝策略上，Ghosted Layers 相比已有 training-free baseline 持续提升 accuracy 和 perplexity，同时保留 layer pruning 的效率收益。

- 🎯 方法：training-free，用小规模 calibration set 推导闭合解线性算子修复 boundary activation 分布偏移
- 💡 创新：证明所提线性算子为 alignment 目标的无约束全局最优，现有方法均为受限子空间解
- 📊 结果：在多个 LLM backbone 和 layer pruning 策略上超越已有 training-free baseline 的 accuracy 与 perplexity
- 🔧 工程：以即插即用模块形式插入存活层边界，不引入推理时额外训练开销
