#### 摘要

- 🎯 用 block diffusion 模型替代自回归 draft 模型，单次 forward pass 并行生成多个 draft tokens
- 💡 draft 模型以 target LLM 提取的 context features 为条件，提升输出质量与 acceptance rate
- 📊 在多种模型与任务上实现超过 6x lossless 加速，较 EAGLE-3 最高提升 2.5x 加速比

#### 📎 相关方法 / 对比基线
