#### 摘要
提出 TAPPA（Temporal Attention Pattern Predictability Analysis）框架，从时间连续视角统一解释 retrieval heads、sink heads、diagonal traces 等碎片化的 attention 模式现象。核心观点是将 attention 模式分为可预测（具明显规律）和不可预测（近似随机）两类，并证明区分依据是 query 沿时间维度的自相似度；对可预测模式从 query、key 与 RoPE 联合作用出发给出三类代表性情况的数学分析。基于该分析提出的简单指标在 KV cache 压缩和 LLM pruning 任务上一致超过 baseline。

- 💡 用 query 时间自相似度统一解释 attention 模式可预测性
- 🎯 联合分析 Q/K/RoPE 推导三类可预测模式的数学形式
- 📊 TAPPA 指标在 KV cache 压缩与 pruning 上均超 baseline
- 🔧 统一解释 retrieval/sink/diagonal heads 等碎片化现象
