#### 🧭 为什么推送这篇
**Survey / 方法论与对比** · topic 3/5 · practicality 3/5 · composite 6/10 · TAPPA是一个分析框架，通过时序视角统一解释多种attention pattern（retrieval head、sink head、diagonal trace等），并将query自相似度作为可预测性指标。该框架并非提出全新的压缩算法，而是对注意力行为提供数学理解，并验证该理解能够改进KV cache压缩和pruning任务的基线。属于measurement/methodology类工作（survey bucket），但不是严格意义上的压缩算法论文。由于最大测试模型未知、benchmark名称未披露、端到端性能提升幅度不明，topic_relevance和practicality均偏中等；代码开源提升可信度，但实际工程价值取决于具体实验规模和benchmark结果。

#### 摘要
针对 LLM 中 attention pattern（retrieval heads、sink heads、diagonal traces 等）观察碎片化、缺乏统一解释的问题，提出 TAPPA（Temporal Attention Pattern Predictability Analysis）框架，从时间连续视角分析 attention 的数学公式，将 pattern 划分为 predictable 与 unpredictable 两类，并指出这种划分可由 query 沿时间维度的自相似度解释。对 predictable pattern，进一步基于 query、key 与 RoPE 的联合作用对三种典型情形做数学推导。基于 TAPPA 导出的简单 metric 应用于 KV cache 压缩与 LLM pruning，均一致优于 baseline。
