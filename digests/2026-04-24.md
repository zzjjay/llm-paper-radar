#### 摘要
SANA-Video 针对长视频生成中 attention 计算和 KV cache 显存随 token 数爆炸的问题，设计了 Linear DiT 与 Block Linear Attention 的常量显存 KV cache 机制：利用 linear attention 的累积性质将历史上下文压缩为固定大小的 state，消除传统 KV cache 的线性显存增长，从而在 RTX 5090 上以 NVFP4 精度高效生成 720×1280 分辨率、分钟级长视频。训练仅需 64 台 H100 GPU 运行 12 天（约为 MovieGen 成本的 1%），在与 Wan 2.1-1.3B、SkyReel-V2-1.3B 同等质量水平下推理延迟快 16×，生成 5 秒 720p 视频从 71s 降至 29s。

- 🎯 Block Linear Attention 利用 linear attention 累积性质构建常量显存 KV cache，支持分钟级长视频生成
- 📊 训练成本仅为 MovieGen 的 1%（64 × H100，12 天）；推理比 Wan 2.1-1.3B / SkyReel-V2-1.3B 快 16×
- 📊 NVFP4 精度在 RTX 5090 上将 5 秒 720p 视频生成从 71s 加速至 29s（2.4× speedup）
- 💡 以固定显存状态替代传统 KV cache，实现 O(1) 显存下的全局上下文建模
