#### 摘要
SANA-Video 针对高分辨率长视频生成的效率瓶颈，提出两项核心设计：用 linear attention 替代 vanilla attention 构建 Linear DiT，以及利用 linear attention 的累积性质设计 constant-memory KV cache 的 block-wise autoregressive 生成方案，从根本上消除长视频生成中 KV cache 随时长线性增长的内存开销。训练成本仅为 MovieGen 的 1%（64 张 H100 GPU 训练 12 天），生成速度比同量级模型 Wan 2.1-1.3B 和 SkyReel-V2-1.3B 快 16×，并可在 RTX 5090 上以 NVFP4 精度部署，5 秒 720p 视频生成延迟从 71s 降至 29s（2.4× 加速）。

- 🎯 方法：Block Linear Attention 以 constant-memory state 实现 KV cache，消除长视频生成的内存随时长增长问题
- 📊 结果：生成速度比 Wan 2.1-1.3B、SkyReel-V2-1.3B 快 16×，训练成本仅为 MovieGen 的 1%
- 🔧 工程：NVFP4 精度部署于 RTX 5090，5 秒 720p 视频生成从 71s 加速至 29s（2.4×）
