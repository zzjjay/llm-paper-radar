- 🎯 方法：用 block diffusion 模型替代 autoregressive draft 模型，单次 forward pass 并行生成 draft tokens
- 💡 创新：draft 模型以 target model 的 context features 为条件，提升生成质量和 acceptance rate
- 📊 结果：跨多模型和任务实现 >6x lossless 加速，比 EAGLE-3 最多快 2.5x
